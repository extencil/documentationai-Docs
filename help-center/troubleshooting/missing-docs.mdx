---
title: Missing Documentation Space
description: Resolve missing doc space issues in Haltman.
---

## Problem

- Can't see your project docs after login.
- "No documentation space found" message.

## Quick Fix

<Steps>
  <Step title="Check project list">
    Ensure you've selected the correct project in the sidebar.
  </Step>
  <Step title="Create a documentation space">
    Click **New Space** and enter a name. Docs will appear under your project.
  </Step>
  <Step title="Refresh the page">
    Reload to update your workspace.
  </Step>
</Steps>
