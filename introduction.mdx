## Overview

Haltman provides a centralized space for your project documentation. You can create, organize, and manage all your docs in one place, making it easy to collaborate and keep everything up to date. Whether you work on software projects, internal wikis, or technical guides, Haltman streamlines the process with intuitive tools and powerful organization features.

This platform supports version control, search, and role-based access, ensuring your documentation stays current and accessible. Start by exploring the key features below to see how Haltman fits your workflow.

## Quick Start

Get up and running with Haltman in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Visit `https://app.haltman.com` and sign up with your email. Verify your account to unlock full features.
  </Step>
  <Step title="Set Up Space" icon="folder-plus">
    Create your first documentation space. Add a name like "My Project Docs" and invite collaborators.
  </Step>
  <Step title="Add Content" icon="edit-3">
    Start a new page with Markdown or our visual editor. Use components like `<Callout>` for highlights.

````markdown
# Welcome Page

Welcome to your project!
````

  </Step>
  <Step title="Publish & Share" icon="share-2">
    Make pages public or share private links. Set up webhooks for notifications on updates.
  </Step>
</Steps>

## Navigation Guide

Learn how to move around the documentation effectively.

<ExpandableGroup>
  <Expandable title="Sidebar Menu" default-open="true">
    The left sidebar shows spaces and pages. Use the search bar at the top for quick jumps. Click the `+` icon to add new content.
  </Expandable>
  <Expandable title="Page Tools">
    Each page has edit, history, and export buttons in the top-right. Use `@mentions` to notify team members.
  </Expandable>
  <Expandable title="Advanced Search">
    Filter by tags, spaces, or date. Advanced queries support boolean operators like `docs AND api`.
  </Expandable>
</ExpandableGroup>
