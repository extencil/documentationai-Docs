## Do I need an account to use the mail forwarding service?

No. The service at [forward.haltman.io](https://forward.haltman.io) does not require registration. You just need to confirm ownership of your destination email.

## Do the OSINT tools require API keys?

It depends on the tool:

| Tool | API Key Required |
|------|:---:|
| ip-thc | No |
| search-leaks | No |
| reverse-whois | Yes (WhoisXML) |
| sub-alter | Yes (WhoisXML) |
| brave-search | Yes (Brave) |

## How do I install the CLI tools?

All tools require **Go 1.22+**. Install via `go install`:

```bash
go install github.com/haltman-io/<tool-name>/cmd/<tool-name>@latest
```

Or clone and build from source. See each tool's documentation page for details.

## How do I report a security issue?

Contact [security@haltman.io](mailto:security@haltman.io). We respond as quickly as possible.
