## Remove an alias

Call the unsubscribe endpoint with your full alias address:

```
https://mail.haltman.io/forward/unsubscribe?alias=<YOUR-ALIAS-ADDRESS>
```

### Example

```bash
curl 'https://mail.haltman.io/forward/unsubscribe?alias=docs.curl%40fwd.haltman.io'
```
