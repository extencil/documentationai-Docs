## Prerequisites

<Callout kind="info" title="Before You Begin">
Ensure you have:
- A modern web browser like Chrome, Firefox, or Safari
- A valid email address for account verification
- Optional: GitHub account for advanced integrations
</Callout>

## Sign Up and Create Your Account

Follow these steps to register and access the Haltman dashboard.

<Steps>
  <Step title="Visit the Signup Page" icon="globe">
    Navigate to the Haltman signup page at `https://app.haltman.com/signup`.
  </Step>
  <Step title="Enter Your Details" icon="user-plus">
    Provide your email, create a strong password, and complete the CAPTCHA if prompted.
    
    Click **Create Account**.
  </Step>
  <Step title="Verify Your Email" icon="mail">
    Check your inbox for a verification email from Haltman and click the link to activate your account.
  </Step>
</Steps>

<Callout kind="tip">
After verification, you'll land on the dashboard. Bookmark `https://app.haltman.com` for quick access.
</Callout>

## Create Your First Documentation Space

Documentation spaces organize your project docs. Set one up in moments.

<Steps>
  <Step title="Click New Space" icon="plus">
    From the dashboard, select **New Space** in the top-right corner.
  </Step>
  <Step title="Configure Space" icon="settings">
    Enter a name like `My Project Docs`, add a description, and choose visibility (private or public).
  </Step>
  <Step title="Save and Enter" icon="folder">
    Click **Create**. You'll enter the space editor ready to add content.
  </Step>
</Steps>

## Upload Your Initial Documents

Add files to your space using these methods.

<Tabs>
  <Tab title="Web Upload" icon="upload">
    Drag and drop files directly into the space editor, or click **Upload** and select Markdown, PDFs, or images.
  </Tab>
  <Tab title="CLI Upload" icon="terminal">
    Use the Haltman CLI for bulk uploads.
    
    ````bash
    npm install -g @haltman/cli
    haltman login --email your@email.com
    haltman space upload my-project-docs ./docs-folder --space-id SPACE_ID
    ````
  </Tab>
  <Tab title="API Upload" icon="code">
    Send documents via the REST API.
    
    <CodeGroup tabs="cURL,JavaScript">
      ````bash
      curl -X POST https://api.haltman.com/v1/spaces/SPACE_ID/documents \
        -H "Authorization: Bearer YOUR_API_KEY" \
        -F "file=@/path/to/doc.md"
      ````
      ````javascript
      const formData = new FormData();
      formData.append('file', fs.createReadStream('/path/to/doc.md'));
      
      fetch('https://api.haltman.com/v1/spaces/SPACE_ID/documents', {
        method: 'POST',
        headers: { Authorization: 'Bearer YOUR_API_KEY' },
        body: formData
      });
      ````
    </CodeGroup>
  </Tab>
</Tabs>
