## External Service Integrations

Our tools integrate with various external APIs and services:

<Columns cols={3}>
  <Card title="ip.thc.org" icon="globe" href="https://ip.thc.org" horizontal="false">
    Public DNS intelligence dataset with 5.14B+ domains. Used by [ip-thc](/open-source/ip-thc). No API key required.
  </Card>
  <Card title="Hudson Rock" icon="shield" href="https://www.hudsonrock.com/" horizontal="false">
    Cavalier OSINT threat-intel API. Used by [search-leaks](/open-source/search-leaks). No API key required.
  </Card>
  <Card title="WhoisXML API" icon="search" href="https://whoisxmlapi.com/" horizontal="false">
    Commercial WHOIS and subdomain data. Used by [reverse-whois](/open-source/reverse-whois) and [sub-alter](/open-source/sub-alter). API key required.
  </Card>
</Columns>

<Columns cols={3}>
  <Card title="Brave Search API" icon="compass" href="https://brave.com/search/api/" horizontal="false">
    Web search API for URL extraction. Used by [brave-search](/open-source/brave-search). API key required.
  </Card>
  <Card title="Postfix" icon="mail" href="https://www.postfix.org/" horizontal="false">
    Production-grade SMTP server. Core of the [mail forwarding stack](/mail-forwarding-selfhost/mail-forwarding-core).
  </Card>
  <Card title="MariaDB" icon="database" href="https://mariadb.org/" horizontal="false">
    Relational database for domain/alias lookups in the forwarding stack.
  </Card>
</Columns>

## API Key Requirements

| Tool | API Provider | Key Required | Free Tier |
|------|-------------|:---:|:---:|
| ip-thc | ip.thc.org | No | N/A (public) |
| search-leaks | Hudson Rock | No | N/A (public) |
| reverse-whois | WhoisXML | Yes | Limited |
| sub-alter | WhoisXML | Yes | Limited |
| brave-search | Brave | Yes | Limited |
